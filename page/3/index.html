  1. 创建一个 img 元素节点；
  2. 设置这个节点的 id属性、src 属性、alt 属性；
  3. 创建一个 p元素节点；
  4. 设置这个节点的 id 属性；
  5. 创建一个文本节点；
  6. 把这个文本节点追加到 p 元素上；
  7. 把 p 元素和 img 元素插入到 gallery.html 中。
