IDEA配置NodeJS

1. 安装 NodeJs

官网下载NodeJS

2. 测试是否成功

控制台输入

node -v

3. 配置工具

  1. 打开IDEA；

  2. 选择file - > setting -> Plugins；

  3. 右边默认没有这个组件，点击 Browe repositories；

  4. 在插件列表搜索 node，看到 Node 插件，点击下载。

  5. 安装完成后重启 IDEA。

4. 试用

重启 IDEA 后，在 create project 里就能看到 node 选项了 。

5. 其他

npm 是随 NodeJS 一起安装的包管理工具。

在终端输入

npm init
npm install jquery –save

加上 –save 将在 package.json 的 dependency 中默认添加 jquery
